# New Project - OpenClaw Workspace

这是一个OpenClaw AI助手的工作空间仓库，包含配置、工具和技能。

## 仓库内容

### 核心配置文件
- `AGENTS.md` - 工作空间指南
- `SOUL.md` - AI助手身份定义
- `USER.md` - 用户信息
- `IDENTITY.md` - 助手身份
- `TOOLS.md` - 工具配置

### 实用工具
- `disable_windows_update.ps1` - Windows更新禁用脚本
- `enable_windows_update.ps1` - Windows更新启用脚本
- GitHub认证和推送工具

### 测试文件
- Memory-merger技能测试
- Python编程记忆测试
- Git工作流记忆测试

## 已安装的技能

这个工作空间配置了以下AI技能：

### 发现与安全
- **find-skills** - 技能查找工具
- **skill-safety-verifier** - 安全验证工具

### 内容处理
- **memory-merger** - 记忆合并工具
- **summarize** - 内容摘要工具

### 网络能力
- **search** - Tavily网络搜索工具

### 开发工作流
- **github-issues** - GitHub Issue管理
- **git-commit** - Git提交优化
- **gh-cli** - GitHub命令行工具
